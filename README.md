# tokenrouter-tools

A CLI tool to set up Codex configuration with your TokenRouter API key.

## Quick Start

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/PaleBlueDot-AI-Open/tokenrouter-tools/main/install.sh | bash -s -- <YOUR_API_KEY>
```

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/PaleBlueDot-AI-Open/tokenrouter-tools/main/install.ps1 | iex; install <YOUR_API_KEY>
```

This will automatically download the appropriate binary, run it with your API key, and clean up afterwards.
